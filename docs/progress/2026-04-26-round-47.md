## Observations

### The drawer-leakage problem and its fix

R46-A intentionally keeps the right drawer open when the fork modal opens — users want to keep browsing node details while deciding. Good UX, but **bad for docs screenshots**: both panels competing for attention makes the hero image look cluttered.

Fix: **do not change the code**. In the browser tool, after clicking "Fork here" and waiting for the modal to mount (`setTimeout(1500)`), inject:

```js
document.querySelectorAll('.ant-drawer').forEach(d => d.style.display = 'none')
```

via `browser_console` just before the screenshot. Modal stays visible, drawer disappears from the image only. Recorded as a skill candidate: `web-docs-screenshot-drawer-hide`.

### AntD modal mount timing

`.ant-modal-content` selector returned `null` immediately after the click in some cases. `[role="dialog"]` was more reliable but also needed ~1500ms for AntD's enter animation to settle. Consistent wait > faster flaky click.

### `scripts/seed_r47a_effects.py` as ground truth

The seed script is committed (197 lines, 3 runs). Anyone (future agent included) can regenerate the exact database that produced these screenshots by running `uv run python scripts/seed_r47a_effects.py --db dogfood.db`. This closes the loop between "the screenshot I see" and "the code that produced it" — no more mystery demos.

## Decisions

1. **R46-A does NOT ship as v0.3.2.** Per CONTEXT.md §5 strategy, bundle R46-A + R46-B + R47-A + R47-B into one alpha. Single-commit patches aren't worth a release, and the fork modal lands more coherently alongside its docs.
2. **Alpha (not v0.4.0 final).** Phase 3 is *mostly* done but not entirely — R48+ may rewrite the AutoGen adapter (currently thin), and the effect-tag badge redesign is still on the table. Alpha reserves that room.
3. **Bilingual guide, English first.** Per user's repeated preference (Chinese to user, English in code/docs), `forking-safely.md` is ~200 lines English + ~190 lines Chinese mirror. TL;DR decision table is at the top so busy readers don't scroll.
4. **README pointer uses GFM reference links.** Matches the existing `[chronos-release-pattern]`-style pattern in the Phase 2 row — consistency over expedience.

## Lessons (added to CONTEXT.md §5 "旧事实仍生效")
