## TL;DR

R8 filled in the v0.1.0 pre-flight gap. Chronos now has **runnable demos
and onboarding docs**, not just a test suite.

- `examples/linear_pipeline.py` — 5-node LangGraph agent (plan → research
  → draft → review → finalize), records baseline, forks at research with
  a `v2-thorough` seed, diff shows exactly which downstream nodes diverged.
  The canonical "Alex" user story from `docs/design/user-stories.md`, now
  actually runnable.
- `examples/router_loop.py` — conditional-edge loop graph, fork overrides
  the round counter to force early exit. Demonstrates that
  `difflib.SequenceMatcher` (ADR-006) handles repeated `node_name`s
  correctly — which the spike 7 validation case 3 already showed, but now
  there's a runnable user-facing demo of it.
- `examples/_fake_llm.py` — deterministic fake LLM (pure function of
  `system, user, seed`). Zero API key / network / flake for demos.
- `docs/getting-started.md` — 5-min onboarding, install → first record →
  first fork → first diff, plus a Troubleshooting section.
- `docs/cli-reference.md` — every command, flag, exit code, env var.
- `CHANGELOG.md` — new, Keep-a-Changelog format.
- `README.md` — rewritten with real install / quickstart / milestone
  table / dev commands. (Was the Phase-0 placeholder from R1.)
- `.gitignore` — excludes `examples/chronos.db` and `*.db-journal`.
- `tests/integration/test_examples.py` — runs both example scripts
  end-to-end and asserts they produce the expected parent+child runs.
