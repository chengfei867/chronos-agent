## Decisions

- **Beta suffix `b0` not minor bump**: we're not at feature-complete for v0.2.0 yet (no diff viewer, no dogfood roundtrip, no GIF). Beta bump signals "try it, expect bugs" to external users while letting us claim the v0.2 theme publicly.
- **CLI status line content pivot**: instead of mechanically replacing `v0.2.0a0` → `v0.2.0b0`, also pivoted the prose content to describe the *current* state (three adapters + Web UI). Status-line drift was the exact R30 skill-pitfall lesson, and stuffing ADR-014 dogfood stats into a 2026-04-24 context was pure sediment.
- **CONTEXT.md §6 plans named R36**, not R35-B/C: R35-A IS this round, so next sub-round is R35-B in principle; but opting to call subsequent work R36-A/B/C recognizes that v0.2.0b0 closes a major chapter and R36 is a fresh planning context. Consistent with R35 = "release round" singleton naming.
- **`[Unreleased]` placeholder text "R35 will decide"**: harmlessly stale (should say "R36 will decide") but not worth a commit amend — will naturally be rewritten when R36 first edit happens.
